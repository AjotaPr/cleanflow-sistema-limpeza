# CleanFlow

Sistema de gerenciamento de serviços de limpeza.

## Funcionalidades
- Cadastro de clientes
- Cadastro de funcionários
- Agendamento de serviços
- Execução com checklist
- Relatórios

## Protótipos
- Baixa fidelidade
- Média fidelidade (Balsamiq)
- Alta fidelidade (Figma)
